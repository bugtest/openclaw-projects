## 决策分类

| 分类 | 描述 | 负责 Agent |
|------|------|------------|
| 代码审查 | PR 审查标准、合并决策 | DevBot |
| 构建策略 | CI 流水线配置、质量门禁 | CIBot |
| 部署决策 | 部署时机、回滚决策 | CDBot |

## 决策流程

1. **问题识别**: Agent 识别需要决策的场景
2. **信息收集**: 收集相关数据和上下文
3. **决策制定**: 基于规则和经验做出决策
4. **执行决策**: 执行决策并记录结果
5. **结果追踪**: 追踪决策效果

## 自动决策规则

### DevBot 自动决策
- PR 审查评分 ≥ 90 → 自动批准
- PR 审查评分 < 60 → 自动请求修改
- 发现高危安全漏洞 → 自动阻止合并

### CIBot 自动决策
- 测试通过率 100% → 构建成功
- 测试通过率 < 100% → 构建失败
- 发现高危安全漏洞 → 构建失败

### CDBot 自动决策
- 健康检查失败 → 自动回滚
- 错误率 > 5% → 自动回滚
- 响应时间 > 2s (持续 1min) → 自动回滚
