## 任务模板

```markdown
### TASK-001
- **类型**: PR 审查
- **Agent**: DevBot
- **优先级**: 高
- **状态**: 待处理
- **描述**: 审查 PR #123
- **创建**: 2024-03-12 10:00
- **完成**: -
```

## 快速命令

- `/task add <description>` - 添加任务
- `/task list` - 查看任务列表
- `/task complete <id>` - 完成任务
- `/task cancel <id>` - 取消任务
