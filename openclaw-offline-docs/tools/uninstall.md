# `openclaw uninstall`

Uninstall the gateway service + local data (CLI remains).

```bash  theme={"theme":{"light":"min-light","dark":"min-dark"}}
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```
